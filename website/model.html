Model Information

Models Used

In our analysis of marketing campaigns, we utilized four different machine learning models to predict and analyze both traditional and online campaign performance:

  • Decision Tree
  • Random Forest
  • Linear Regression
  • K-Nearest Neighbors (KNN)

Model Evaluation

Each model was evaluated using the following metric:

  • RMSE (Root Mean Square Error): Measures the standard deviation of the residuals (prediction errors). Lower values indicate better fit.
